**git init:** _Khởi tạo một Git repository trống ngay tại thư mục hiện hành._

**git clone <url>:** _Sao chép một repository từ GitHub về máy tính._

**git add <file>:** _Đưa một file cụ thể vào khu vực chờ (Staging Area)._

**git add .:** _Đưa tất cả các file đã thay đổi vào khu vực chờ._

**git commit -m "Thông điệp":** _Lưu vĩnh viễn các thay đổi vào lịch sử kèm theo mô tả._

**git branch -a:** _Liệt kê danh sách các nhánh hiện có._

**git branch <ten-nhanh>:** _Tạo một nhánh mới nhưng chưa chuyển sang._

**git checkout <ten-nhanh> hoặc git switch <ten-nhanh>:** _Chuyển sang nhánh khác._

**git push origin <ten-nhanh>:** _Đẩy code từ máy lên GitHub_

**git checkout -- <file>:** _Hủy bỏ các thay đổi vừa thực hiện trên một file (quay về trạng thái lúc chưa sửa)._

**git reset --hard HEAD:** _Xóa bỏ toàn bộ thay đổi chưa commit, đưa dự án về trạng thái của lần commit gần nhất._

**git stash:** _Tạm thời cất các thay đổi đang làm dở để chuyển sang nhánh khác mà không cần commit._
